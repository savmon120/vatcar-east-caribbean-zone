# Dropdown menu

A dropdown menu is a toggleable menu that allows the user to choose one value from a predefined list.

It is supported to use the special urls `<button>`, to display keyboard-accessible link text only and
`<nolink>` to display link text only.
Also, to add a menu divider set the dropdown menu item text as "-" and url as `<nolink>`.

## Usage

A practical example is in the **Nav** component:

```twig
{%
  include 'radix:dropdown-menu' with {
    items: item.below
  }
%}
```
